# BLACK IP - Resenha|Cyber
    Black List para o CLI

    Tool Reformulada do Adionditsak | blacklist-check-unix-linux-utility - Criada em 2014

### Instalação

    $git clone https://github.com/resenhacyber/blackip.git
    $cd blackip
    $chmod +x blackip
    $mv blackip /usr/bin
